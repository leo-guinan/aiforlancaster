## What this is

This repo contains everything behind aiforlancaster.com:

- A complete book manuscript (~40,000 words) on AI for small business
- 27+ guide pages covering specific industries and use cases
- A static site builder that converts markdown → HTML with auto-linking and sitemaps
- A daily guide generator that uses Claude to produce new content on a schedule
- A trends agent that monitors rising searches in your area (Columbus DMA / Ohio)
- Deployment config for Cloudflare Pages

**The whole thing is designed to be forked.** Replace Lancaster with your town, swap the local references, deploy. See [`fork/`](fork/README.md) for instructions.

## Quickstart — run it locally

```bash
git clone https://github.com/leo-guinan/aiforlancaster
cd aiforlancaster

# Install dependencies
pip3 install pytrends certifi

# Build the site
python3 scripts/build_site.py

# Preview deploy/ in any local server
cd deploy && python3 -m http.server 8000
```

---

## Deploying

The site deploys to Cloudflare Pages. If you fork this, create a new Pages project and point it at your `deploy/` directory.

```bash
npx wrangler pages deploy ./deploy --project-name=your-project-name
```

## The pipeline

Three scripts do all the work. See [`scripts/README.md`](scripts/README.md) for full documentation.

| Script | What it does |
|--------|-------------|
| `build_site.py` | Converts all guide markdown → HTML, rebuilds homepage, generates sitemap, inserts internal links |
| `generate_guide.py` | Generates a new guide page using Claude, then calls build_site.py |
| `trends_agent.py` | Weekly pulse: rising searches in your area + Reddit signal |

## Forking for your town

See [`fork/README.md`](fork/README.md) — full instructions.

The short version:
1. Fork this repo
2. Edit `town-config.yaml` with your town's details
3. Edit `landing-page/index.html` with your name and contact
4. Replace Lancaster-specific content in `book/` and `site/guides/`
5. Run `python3 scripts/build_site.py`
6. Deploy

The book content is ~80% universal. The other 20% is local references — business names, county resources, chamber info. That's the part you replace.
